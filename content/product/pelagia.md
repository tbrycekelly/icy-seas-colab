# Pelagic imagery to oceanographic data

Pelagia supports the work between raw pelagic imagery and quantitative analysis: ingestion, detection, review, measurement, labeling, quality control, model development, and export.

<div class="portfolio-meta">
<p><strong>Pelagia:</strong> Open-source analysis server</p>
<p><strong>PelagiaView:</strong> Closed-source web GUI</p>
<p><strong>Status:</strong> Active development with a public demonstration</p>
<p><strong>Best for:</strong> Large optical-oceanography image collections that require scalable processing and scientific review</p>
</div>

## One connected suite

The backend and web interface are separate by design. Pelagia can run close to image data and processing resources, while PelagiaView gives scientists an interactive browser workspace for review, quality control, and export.

Together they support:

- Video and large-frame ingestion
- Preprocessing, segmentation, and ROI refinement queues
- Background CPU and GPU workers
- Frame inspection with detections and surrounding image context
- ROI, mask, measurement, and classification review
- Durable metadata and content-addressed artifacts
- Reproducible, analysis-ready data products

## Learn more and try it

Pelagia has its own website with a detailed workflow overview, example imagery, application notes, and a live demonstration. Backend source is available through the Icy Seas GitHub organization; PelagiaView is commercial closed-source software.
